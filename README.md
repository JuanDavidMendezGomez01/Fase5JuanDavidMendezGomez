# Fase 5 - Publicación del sitio Web en un repositorio en línea

## 📌 Descripción del Proyecto
Este proyecto consiste en un sitio web interactivo y adaptable enfocado en la **Arquitectura Japonesa**, desarrollado para el curso de **Programación Web** (Código: 213025) de la UNAD. 

A lo largo de las distintas fases del componente práctico, la interfaz evolucionó incorporando:
*   **Fase 2:** Maquetación semántica estructurada en HTML5 y hojas de estilo CSS3 con un diseño minimalista basado en la cultura nipona.
*   **Fase 3:** Implementación de diseño adaptable (Responsive Design) mediante Flexbox y Media Queries, logrando adaptabilidad multiplataforma y una calificación del 100% en accesibilidad bajo estándares WCAG mediante Lighthouse.
*   **Fase 4:** Integración de elementos dinámicos mediante JavaScript (manipulación del DOM para saludos basados en la hora del sistema, menús desplegables con eventos de ratón y un carrusel de imágenes automatizado).
*   **Fase 5:** Despliegue, control de versiones y publicación en la nube.
